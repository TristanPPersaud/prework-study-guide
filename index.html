HTML

  • Hypertext Markup Language (HTML) is one of three cornerstone programming languages.
  • HTML creates the structure of the webpage.
  • Webpage is comprised of types of content called elements.
  • Common elements which add content include: Text, Image, Header, and List.
  • Common elements which divide and describe webpage: Meta-related elements, Content division elements, and Semantic elements.
  • Elements are contained between an opening and closing tag.
  • Element refers to everything between opening and closing tag. Tag refers to what is inside brackets.
  • The head element contains information about the webpage.
  • The meta element contains information about metadata such as the character set, description, keywords, author, and viewport.
  • Metadata is information the browser uses to provide our webpage to users. Metadata is not seen by users.
  • The charset element defined by the head element sets programmers character encoding.
  • The http-equiv element defined by the head element sets an HTTP header for content.
  • The viewport element sets the default view for a programmers webpage.
  • The title element defines the title for a webpage which is important for Search Engine Optimization (SEO)
  • The body element represents the visible content shown to the user.
  • Elements defined by the body element are called "child elements".
  • Semantic HTML elements describe its content and the role it plays on a webpage.
  • Semantic elements make HTML easier to read by developers and browsers.
  • The body element is a semantic element.
  • Non-semantic elements include div to create a division in the page or span to create an inline division.
  • The section element defines a section in a document that holds thematic content.
  • The h1,h2,h3,h4,h5,h6 elements represent the heading (larger text) of a website.
  • The number in the heading element determines the size of heading, 1 being the largest and 6 the smallest.
  • The p element represents a paragraph or passage. The p element contains text on webpages.
  • The ul element (unordered list), ol element (ordered list), and li element (list item) are used to create and represent lists.
  • The img element containes information about images on webpage. Contains defining attributes like src, alt, width, and height.
  • The src attribute defines location of image.
  • The alt attribute defines the image.
  • The width attribute defines width of image.
  • The height attribute defines the height of an image.
  • The br element creates a line break between content.
  • The a (anchor) element creates link to webpages.
  • The href attribute includes the URL being reached by the a element.
  • The footer element is at the bottom of the page and contains the author, copyright, contact, sitemap, and navigation.

CSS

  • Cascading style sheets, (CSS), is a programming language that is used to define webpage content presentation.
  • Three ways to style a webpage include inline CSS, internal CSS style sheet, and external CSS stylesheet.
  • Inline CSS applys stle to a specific HTML tag.
  • Internal style sheet implements an entire style sheet into the HTML file by using he style element in the head element.
  • External style sheet is linked into the HTML seperating the CSS rules to another file.
  • External style sheets are linked through the link element placed in the HTML head element.
  • The link element has a rel component, defining what is being linked, and an href component, the actual link to the content.
  • A CSS rule is comprised of a selector and a declaration made up of a property and its chosen value.
  • The * is used as a selector to apply a CSS rule to all elements on a page.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The text align property is a positioning property and can be used to center text.
  • The img selector is used to assign property values to ant images.
  • Property values may also be edited for lists, ul, and text, p.
  • Class selectors may be assigned to HTML elements.
  • Class selectors allow for any assigned HTML elements to have property values applied to them as a group.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git checkout main: switches you to the main branch.
  • git add -A: adds current branch to staging.
  • git config: assists in configuring global ID for email adresses, user names, etc.
  • git flow is the process of saving and commiting edits to files from the local repository to the remote repository.
  • first add -A commits all files in working branch, then git commit -m "name" to commit files to remote repository, then git pull origin main to make sure current branch is up to date syncing repositories, then git push origin feature/"name" to push all changes to github repository, finally create pull request on github and merge branches.

JavaScript

  • Javascript is a programming language that is used to add interactivity to websites.
  • Javascript must be linked to the index.html file.
  • Javascript is linked just before the body wraps through script src = ("java file name").
  • A variable is a named container used for storing data in code.
  • The const keyword creates a variable that is permanent and can not be reassigned.
  • After the const keyword a variable is given a unique name used to referance information stored in variables.
  • The assignment operator = is used to assign data to be stored in the variable.
  • A string is a variable that includes multiple pieces of data.
  • Strings are always contained in quotation marks.
  • Every line of javascript code ends in a semicolon.
  • The console.log method outputs messages to the web console.
  • Chrome Dev tools can be used to view elements and console log in browser.
  • Control flow is the order in which a computter executes code. Java runs from the first statement at the top to the last at the bottom of script.
  • Conditional statements and loops can be used to alter control flow.
  • An if statement introduces decision making to code. Code will only execute if conditions are met or is true.
  • Javascript checks to see if condition is truthy, if considered truthy it will execute the code.
  • Javascript considereds a value falsy if it contains the value 0, -0, On, "", null, undefined, Nan, or a Boolean data with the value false.
  • Else if can be used to provide more conditions to the computer.
  • The strict equality operator, ===, sees if two values are equal and returns a Boolean result either true or false.
  • An array is a group of related data in a variable.
  • Index is a unique number given to the location of a data item in an array.
  • A for loop uses a predictable pattern of indices to perform a task an all array items using a single code block executed repeatedly.
  • The length of array lets the for loop know how many times to run code based on the ammount of items in array.
  • Length of array can be determined by using length.
  • The let keyword creates a variable thats value can be reassigned.
  • A for loop contains three statements.
  • First statement determines the starting point for the loop. Declare a variable and assign a number to it.
  • Second statement is the condition. Loop keeps running until condition is met.
  • Third staetement allows array to iterate over each item.
  • Commenting out code makes it inactive without deleting the code. Use /* and */ at the beginning and end of code.
  • A function is a set of instructions that tells the computer how to achieve certain tasks.
  • Two requirements exist for a Javascript function. One define the function, and two call the function.
  • Code block or function body instructs the computer on how to carry out the function.
  • Declare a function using the function keyword.
  • To call a function input the functions name.
  • A function's name will always be followed by parantheses.
  • The code for a functions task is added inside of curly brackets {} after the functions name.
