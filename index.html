HTML

  • Hypertext Markup Language (HTML) is one of three cornerstone programming languages.
  • HTML creates the structure of the webpage.
  • Webpage is comprised of types of content called elements.
  • Common elements which add content include: Text, Image, Header, and List.
  • Common elements which divide and describe webpage: Meta-related elements, Content division elements, and Semantic elements.
  • Elements are contained between an opening and closing tag.
  • Element refers to everything between opening and closing tag. Tag refers to what is inside brackets.
  • The head element contains information about the webpage.
  • The meta element contains information about metadata such as the character set, description, keywords, author, and viewport.
  • Metadata is information the browser uses to provide our webpage to users. Metadata is not seen by users.
  • The charset element defined by the head element sets programmers character encoding.
  • The http-equiv element defined by the head element sets an HTTP header for content.
  • The viewport element sets the default view for a programmers webpage.
  • The title element defines the title for a webpage which is important for Search Engine Optimization (SEO)
  • The body element represents the visible content shown to the user.
  • Elements defined by the body element are called "child elements".
  • Semantic HTML elements describe its content and the role it plays on a webpage.
  • Semantic elements make HTML easier to read by developers and browsers.
  • The body element is a semantic element.
  • Non-semantic elements include div to create a division in the page or span to create an inline division.
  • The section element defines a section in a document that holds thematic content.
  • The h1,h2,h3,h4,h5,h6 elements represent the heading (larger text) of a website.
  • The number in the heading element determines the size of heading, 1 being the largest and 6 the smallest.
  • The p element represents a paragraph or passage. The p element contains text on webpages.
  • The ul element (unordered list), ol element (ordered list), and li element (list item) are used to create and represent lists.
  • The img element containes information about images on webpage. Contains defining attributes like src, alt, width, and height.
  • The src attribute defines location of image.
  • The alt attribute defines the image.
  • The width attribute defines width of image.
  • The height attribute defines the height of an image.
  • The br element creates a line break between content.
  • The a (anchor) element creates link to webpages.
  • The href attribute includes the URL being reached by the a element.
  • The footer element is at the bottom of the page and contains the author, copyright, contact, sitemap, and navigation.
